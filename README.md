<h1 align="center">🧮 System of linear equations calculator 🧮</h1>

## 🚀 Introduction

This project is an implementation in plain C of different methods to solve systems of linear equations :
- (PA)LU decomposition
- Gauss elimination
- Cholesky decomposition
- Gauss-Seidel method
- Jacobi method
