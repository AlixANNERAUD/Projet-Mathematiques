<h1 align="center">🧮 System of linear equations calculator 🧮</h1>

## 🚀 Introduction

As part of our mathematics course, we had to implement different methods to solve systems of linear equations in plain C :
- (PA)LU decomposition
- Gauss elimination
- Cholesky decomposition
- Gauss-Seidel method
- Jacobi method
